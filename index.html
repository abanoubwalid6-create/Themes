🗑️ مسح تطبيقات شاومي (Bloatware)

مسح تطبيق موسيقى شاومي:

pm uninstall -k --user 0 com.miui.player

مسح تطبيق مشغل الفيديو:

pm uninstall -k --user 0 com.miui.videoplayer

مسح تطبيق المتصفح:

pm uninstall -k --user 0 com.mi.globalbrowser

⚡ تحسين الأداء (بدون روت)

منع خنق المعالج (Throttling):

settings put global power_manager_constants "no_cached_processes_limit=true"

تسريع فتح التطبيقات (FSTRIM):

settings put global fstrim_mandatory_interval 3600

إجبار GPU على معالجة الواجهة:

settings put global hardware_acceleration_enabled 1

🎮 الحساسية والشاشة

حساسية اللمس القصوى (Free Fire Mode):

settings put system pointer_speed 7

تثبيت 120Hz إجباري:

settings put system peak_refresh_rate 120.0
تم النسخ بنجاح! ✅
